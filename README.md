# Where's Waldo (Nintendo Edition)

### [Live Demo](https://skukhniy.github.io/nintendo-wheres-waldo/)

## Getting started

After cloning the repository, run:

```elm
npm install
npm start
```

The site is now running at `localhost:3000`.
